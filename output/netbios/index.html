NetBIOS

NetBIOS is a networking protocol that enables communication between computers on a local area network (LAN).

Description

NetBIOS, which stands for Network Basic Input/Output System, is a protocol that was developed in the 1980s to allow applications on different computers to communicate within a local area network (LAN). Originally designed for IBM PC networks, it provides services related to the session layer of the OSI model, allowing applications to communicate over a network through session establishment, maintenance, and termination. NetBIOS allows for the identification of computers and services on a network using simple names, facilitating file sharing and printer access. However, due to its original design limitations and security vulnerabilities, NetBIOS has been largely replaced by more robust protocols such as TCP/IP. In the context of network security, NetBIOS can pose risks, as it may expose systems to various attacks, including name resolution spoofing and information disclosure. Organizations are encouraged to disable NetBIOS over TCP/IP in environments where it is not needed, as it can be leveraged by attackers to gather information about a network.

Examples

  • File sharing between Windows computers using NetBIOS names.
  • Accessing shared printers in a LAN environment using NetBIOS protocols.

Additional Information

  • NetBIOS operates primarily over TCP/IP using ports 137, 138, and 139.
  • It is often used in legacy systems and still found in some enterprise environments.
